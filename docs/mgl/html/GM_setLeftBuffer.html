GM_setLeftBuffer

Set active rendering to the left stereo buffer.

Declaration

void MGLAPI GM_setLeftBuffer(void)

Prototype In

gm/gm.h

Description

This function sets the active rendering buffer for all subsequent drawing commands to the left stereo buffer.
