EVT_getCodePage

Returns the currently active code page for translation of keyboard characters.

Declaration

codepage_t * EVTAPI EVT_getCodePage(void)

Prototype In

event.h

Return Value

Pointer to the currently active code page translation table.

Description

This function is returns a pointer to the currently active code page translation table. See EVT_setCodePage for more information.

See Also
