LZTimerOn

Starts the Long Period Zen Timer counting.

Declaration

void ZAPI LZTimerOn(void)

Prototype In

ztimer.h

Description

Obsolete function. You should use the LZTimerOnExt function instead which allows for multiple timers running at the same time.
