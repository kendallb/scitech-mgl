MGL_bitBlt

Blts a block of image data from one device context into another.

Declaration

void MGL_bitBlt(
    MGLDC *dst,
    MGLDC *src,
    rect_t r,
    int dstLeft,
    int dstTop,
    int op)

Prototype In

mgraph.h

Parameters

dst

Destination device context

src

Source device context

r

Rectangle defining are to Blt from

dstLeft

Left coordinate to Blt to

dstTop

Right coordinate to Blt to

op

Write mode to use during Blt

 

Description

This function is the same as MGL_bitBltCoord, however it takes entire rectangles as parameters instead of coordinates.
