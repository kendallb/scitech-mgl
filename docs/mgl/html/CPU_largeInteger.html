CPU_largeInteger

Declaration

typedef struct {
    ulong   low;
    ulong   high;
    } CPU_largeInteger

Prototype In

cpuinfo.h

Description

Defines the structure for holding 64-bit integers used for storing the values returned by the Intel RDTSC instruction.

Members

low

Low 32-bits of the 64-bit integer

high

High 32-bits of the 64-bit integer
